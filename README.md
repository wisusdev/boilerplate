## 🔧 Dev Installation


```bash 
  git clone https://github.com/wisusdev/boilerplate.git

  cd boilerplate

  cp .env.example .env

  composer install

  php artisan key:generate

  php artisan migrate --seed

  php artisan storage:link
  
  php artisan serve
```
